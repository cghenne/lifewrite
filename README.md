# LifeWrite

LifeWorks Hackathon

# Setup
* install docker ()
* run docker application (osx and windows only)

Setup all dependencies

```
./dicker up
```

Login to the SSH

```
./dicker ssh
```

## Client Instructions

From under terminal: 
- `npm install` (once)

- `npm run client` to start the client development build at [http://localhost:3000](http://localhost:3000)


## API Instructions

- `npm run server`


## build

- `npm run build`
